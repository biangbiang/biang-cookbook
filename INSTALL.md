# 运行环境配置

1. 安装NodeJS（建议采用v10.15.3）

2. 安装gitbook（建议采用 GitBook version:3.2.2，CLI version: 2.3.2）

  ```
  npm install gitbook-cli -g
  gitbook fetch 3.2.2（如果版本不是3.2.2，执行这个命令修改gitbook版本）
  gitbook install
  ```

3. 运行项目

  ```
  gitbook serve
  ```

4. 生成静态文件

  ```
  gitbook build
  ```

5. 浏览器访问

  ```
  http://localhost:4000
  ```
